# System Manifest

## Goal
Build a card-based investment research system that turns raw financial content into reusable research units.

## Core principles
- Raw data is stored once.
- Different tasks use different distillation prompts.
- Opportunity line and learning line are separated.
- Every higher-level card must be traceable to source cards.
- Counterviews must be preserved.
- Rules and hypotheses must be verifiable or falsifiable.

## Primary card types
- raw_source
- episode
- insight
- counterview
- research_topic
- hypothesis
- rule
- validation

## Closed loop
1. raw_source
2. episode extraction
3. insight extraction
4. counterview extraction
5. topic promotion
6. hypothesis generation
7. validation
8. rule update
9. decision support
